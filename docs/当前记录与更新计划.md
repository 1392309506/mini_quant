## 三、风险警示

### 高杠杆 ≠ 安全回撤

回测中的 -30% 回撤在真实杠杆交易中意味着极高的爆仓风险：

以 `p=50, top_k=5` 为例：
- `position_size = 1/5 × 50 = 10` → 每个仓位 = 账户的 **10 倍**
- 同时最多 5 个仓位 → **总敞口 50 倍**
- **1 个仓位反向 10%** = 账户归零
- **2% 反向** = 亏损 20%

真实交易中：
- Exness CFD 保证金要求通常 1%~2%
- 亏损达到保证金的 50%~80% 即触发强制平仓
- **实际可承受亏损可能只有账户的 1%~2%**

**安全杠杆估算（已验证）**:
- 基于最差单日（scale=1.0 时 -0.29%）：`max_scale = 0.02 / 0.0029 ≈ **7x**`
- 基于 3σ 日波动（0.03%/日）：`max_scale = 0.02 / (3×0.0003) ≈ **20x**`
- **推荐保守值: ≤ 7x**

**费用模型验证**: vectorbt 在 `cash_sharing` 模式下费用计算正确（费用与杠杆等比例放大），已验证 scale=1x~50x 下损耗率约 1.6%。之前的"Fees=0"误解源于 `pf.fees()` 报告值为 0 的显示问题，实际费用已正确计入回测收益。详见[验证结果](#七回测结果有效性验证)。

#### ⚠️ 重要发现：模型收益的主要驱动力

- **主要驱动力是重新平衡时机**（在牛市中频繁买入），而非个股选择能力

- **需要更多横截面区分度**来提高个股选择带来的超额收益
