## 命令速查

| 命令 | 作用 |
|---|---|
| `python data_fetcher.py` | 增量更新（缓存 ≤7 天则跳过） |
| `python data_fetcher.py --force` | 强制重新下载 |
| `python data_fetcher.py --max 5` | 只下载前 5 个标的（测试用） |
| `python factor_engine.py` | 计算全量因子 |
| `python factor_engine.py --cache` | 用缓存数据计算因子 |
| `python train_model.py` | 完整 Walk-Forward 训练 |
| `python train_model.py --quick` | 快速模式（5 个标的） |
| `python train_model.py --target 21` | 预测 forward_return_21 |
| `python train_model.py --load <exp_id>` | 查看已有实验摘要 |
| `python run_backtest.py <exp_id>` | 基于实验运行回测 |
| `python run_backtest.py <exp_id> --top 10` | 回测时覆盖 top_k 参数 |
| `python run_backtest.py <exp_id> --no-regime` | 关闭市场状态过滤器 |

### 数据完整性报告

```
检查: OK=22(85%), 需关注=4/26
  [V] 数据偏少: 420天 (<200，无法算 MA200)
  [BRK-B] 缺失值: 45个(2.3%)
```

- **数据偏少**：该股票上市较晚，不足 200 天。可从 `TRADE_UNIVERSE` 移除，或不依赖它的 MA200 因子。
- **异常波动 >±20%**：可能是财报跳空或拆股。少量可接受，大量需检查数据。

### 因子验证

```

```

通常源于：财报日跳空、拆股未还原、数据源错误。在 Jupyter 里定位具体日期排查。

## 常见问题

### Q1: yfinance 报 "Too Many Requests / Rate limited"

**原因：** 代理未开启，或代理端口填错。
**解决：**
1. 确认 v2rayN/Clash 正在运行
2. 在 `.env` 中确认 `HTTP_PROXY` 端口与代理软件一致
3. 测试代理是否通：
   ```bash
   curl -x http://127.0.0.1:10808 https://www.google.com -I
   ```

### Q2: 不想用代理，能用 AV 吗？

能，但免费层 AV 只返回最近 100 天数据，不足以计算 200 日均线因子。仅作兜底：

```ini
# .env
DATA_BACKEND=alpha_vantage
```

### Q3: "ModuleNotFoundError: No module named 'quant'"

从项目根目录运行脚本，Python 才能找到 `quant/` 包：

```bash
cd d:/Code_store/Quant
python data_fetcher.py
```

### Q4: 因子全是 NaN

某股票数据不足 60 天会被跳过。检查 `data/factor_validation.csv`，从 `quant/universe.txt` 移除问题标的。

## Jupyter 高级用法

```python
from src.data.fetcher import fetch_all_data, extract_close_matrix, extract_volume_matrix
from src.factors import build_factor_panel, rsi_14

df = fetch_all_data()
closes = extract_close_matrix(df)
volumes = extract_volume_matrix(df)

factors = build_factor_panel(closes, volumes)

# 查看 AAPL 最新 RSI
aapl_rsi = factors["AAPL"]["RSI_14"].dropna()
print(f"AAPL RSI: 最新={aapl_rsi.iloc[-1]:.1f}")

# 单因子分析
rsi = rsi_14(closes["AAPL"])
oversold = rsi[rsi < 30]   # 超卖信号
print(oversold.tail(5))
```
