Exness 的真实账户主要有这几类：Standard、Standard Cent、Pro、Raw Spread、Zero、Crypto。你做的是量化系统（因子→模型→回测→自动下单），核心诉求是点差低、允许算法交易/EAs、执行快不 requotes。我的建议：

首选 Raw Spread（点差 0 起 + 佣金 $3.5/手/边）

点差几乎为零，适合量化里高频/剥头皮/网格类策略，成本可预测（成本 = 固定佣金，方便回测里精确建模）
明确允许算法交易和 EAs
Market Execution，不会 requotes
次选 Zero（前 30 个主流品种 0 点差 + 佣金），如果你主要交易主流外汇对/黄金，它和 Raw Spread 接近，有时零点差品种覆盖更稳。

不推荐：

Standard / Standard Cent：点差最宽（0.3 起），适合新手手动交易或练手，量化高频交易成本会被吃掉；Cent 账户纯粹是美分练手用
Pro：无佣金但点差 0.1 起，比 Raw 贵，且执行方式对算法不友好；只有你策略频率很低（几天一次）时才考虑
