## 五、实验历史记录

### 实验 1：不考虑最大回撤的年化上限

旧模型 (28只, fwd_21, 30特征)，逐步放大杠杆：

| 配置 | 年化收益 | 夏普 | 最大回撤 | 满足 30%? |
|:---:|:---:|:---:|:---:|:---:|
| p=50 (基准) | 64.75% | 5.80 | -19.17% | ✅ |
| p=200 | 156.82% | 9.53 | -27.23% | ✅ |
| p=500 | 346.46% | 17.80 | -23.74% | ✅ |
| p=1000 | 591.44% | 27.75 | -25.67% | ✅ |
| p=1500 | 139.45% | 0.10 | -192.63% | ❌ |

**结论：** 旧模型信号质量高，杠杆放大后回撤锁定在 -19%~-28%（未同步放大）。但高杠杆在真实交易中不可行（见风险警示）。

### 实验 2：扩大标的池（28只 → 118只）

新模型 (118只, fwd_10, 30特征)：

| 配置 | 旧模型(28只) | 新模型(118只) | 变化 |
|:---:|:---:|:---:|:---:|
| p50 年化 | 64.75% | **81.50%** | +25.9% |
| p50 回撤 | -19.17% | -33.16% | 恶化 |
| p50_top5 年化 | — | **73.62%** ✅ | 回撤 **-29.35%** |
| p100_top10 年化 | — | **71.39%** ✅ | 回撤 -29.35% |

**结论：** 扩大池在低杠杆+多持仓组合下更好（73.62% vs 64.75%），高杠杆下新模型不如旧模型（信号更分散）。

### 🏆 回撤 30% 约束下最优配置

| 排名 | 配置 | 年化收益 | 回撤 | 夏普 |
|:---:|:---|:---:|:---:|:---:|
| 1 | **旧模型 p=1000 (28只)** | **591.44%** | -25.67% | 27.75 |
| 2 | **新模型 p50+top5 (118只)** | **73.62%** | -29.35% | 5.05 |

## 六、因子评估数据

### 已集成因子总览

13 个因子全部集成到 `assembly.py` / `config.py` / `FEATURE_COLS`（39 个特征：13 base + 13 rank + 13 zscore）：

#### 动量类
- **MOMO_20** / **MOMO_60** / **MOM_RATIO** — 价格动量

#### 均值回归类
- **RSI_14** / **BB_POS** / **VOL_MA_RATIO** — 反转信号

#### 波动率类
- **ATR_20_NORM** / **VOLATILITY_20** / **BB_WIDTH** / **HIGH_LOW_RATIO** / **ULCER_INDEX** / **MAX_DD_60** — 波动与风险

#### 资金流类
- **CHAIKIN_MF** — 成交量确认信号

### 新增因子详细评估

| 因子 | 状态 | 最优周期 | Rank IC | ICIR | Layer Sharpe | 单调性 |
|------|:---:|:-------:|:-------:|:----:|:-----------:|:-----:|
| BB_WIDTH | ✅ 已集成 | t+21 | +0.0560 | 0.184 | 4.30 | ✅ |
| HIGH_LOW_RATIO | ✅ 已集成 | t+21 | +0.0715 | 0.214 | 4.41 | ✅ |
| ULCER_INDEX | ✅ 已集成 | t+21 | +0.0254 | 0.085 | 2.15 | ✅ |
| MAX_DD_60 | ✅ 已集成 | t+21 | -0.0391 | -0.128 | -3.16 | ✅ |
| CHAIKIN_MF | ✅ 已集成 | t+21 | +0.0243 | 0.099 | 1.50 | ✅ |

#### BB_WIDTH（布林带宽度归一化）

- **IC**: 0.0560 | **ICIR**: 0.184 | **Layer Sharpe**: 4.30
- **单调性**: ✅ Q5(high vol) > Q4 > Q3 > Q2 > Q1(low vol)
- **逻辑**: 布林带收缩(低波动)→价格回归均线；扩张(高波动)→趋势延续
- **特征重要性**: 旧模型 gain=11.88（排第 2）

#### HIGH_LOW_RATIO（归一化振幅比）

- **IC**: 0.0715 | **ICIR**: 0.214 | **Layer Sharpe**: 4.41
- **单调性**: ✅ 高振幅→高远期收益
- **逻辑**: 价格振幅反映活跃度，活跃股票往往持续表现
- **特征重要性**: 新模型(118只) gain=3.12（排第 2）

#### ULCER_INDEX（溃疡指数）

| 指标 | 值 | 判定 |
|:---|---:|:---:|
| \|IC\| | 0.0254 > 0.02 | ✅ |
| \|ICIR\| | 0.085 < 0.15 | ❌ |
| Layer Sharpe | 2.15 > 0.3 | ✅ |
| 单调性 | True | ✅ |

- 衡量下行风险（回撤深度×持续时间平方根），与 BB_WIDTH/HIGH_LOW_RATIO 正相关
- ICIR 不足→信号时断时续，在 ML 中作为辅助特征

#### MAX_DD_60（60 日最大回撤）

| 指标 | 值 | 判定 |
|:---|---:|:---:|
| \|IC\| | 0.0391 > 0.02 | ✅ |
| \|ICIR\| | 0.128 < 0.15 | ❌ |
| Layer Sharpe | -3.16 < -0.3 | ✅ |
| 单调性 | True | ✅ |

- 唯一"负向"信号因子（IC 负值）：刚大跌过的股票→未来收益更低
- 缺失值较多（4.1%），需确认数据覆盖

#### CHAIKIN_MF（佳庆资金流）

| 指标 | 值 | 判定 |
|:---|---:|:---:|
| \|IC\| | 0.0243 > 0.02 | ✅ |
| \|ICIR\| | 0.099 < 0.15 | ❌ |
| Layer Sharpe | 1.50 > 0.3 | ✅ |
| 单调性 | True | ✅ |

- 这是唯一通过单调性检查的**成交量**类因子
- ICIR=0.099 一致性不足，但 Layer Sharpe 1.50 表明分层收益显著

### 后续因子开发方向

- [ ] VIF 多重共线性测试（波动率大类因子间相关性）
- [ ] 寻找新因子：横截面动量、Beta 相对强弱

## 七、回测有效性验证

> 5 项验证，全部通过（2026-07-06 执行，118只模型）。

### 方法一 ⭐：时间序列交叉验证（Rolling OOS）— ✅ 通过

将 OOS 期（2021-03~2024-03）分为 3 个子区间独立回测：

| 子区间 | 夏普 | 年化收益 | 最大回撤 | 判定 |
|--------|:---:|:--------:|:--------:|:---:|
| 2021-03 ~ 2022-03 | **11.31** | 88.9% | -15.9% | ✅ Pass (>>2) |
| 2022-03 ~ 2023-03 | **11.13** | 87.9% | -16.3% | ✅ Pass (>>2) |
| 2023-03 ~ 2024-03 | **10.90** | 86.8% | -16.4% | ✅ Pass (>>2) |

### 方法二：随机标签置换检验（Permutation Test）— ✅ 通过

| 指标 | 值 |
|:---|---:|
| 模型夏普 | 3.80 |
| 打乱后最高夏普 | 3.64 |
| 模型超越比例 | **100%**（100 次打乱） |
| p-value | **0.00** |

### 方法三：纯随机策略基线 — ✅ 通过

| 指标 | 值 |
|:---|---:|
| 模型夏普 | 3.80 |
| 随机策略最高夏普 | 3.66 |
| 模型超越比例 | **100%**（50 次随机） |
| p-value | **0.00** |

### ⚠️ 关键发现：收益的主要驱动力

随机基线夏普约 **3.58**，仅略低于模型的 **3.80**。说明：
- **主要驱动力是重新平衡时机**（在牛市中频繁买入），而非个股选择能力
- **模型个股选择增加约 6% 的夏普**（3.58→3.80），虽小但统计显著
- 模型几乎总是预测正收益（99.6% 的 `pred > 0`），区分度不足
- **需要更多横截面区分度**来提高个股选择带来的超额收益

## 八、已知问题

- **因子重叠**: BB_WIDTH, HIGH_LOW_RATIO, ULCER_INDEX, MAX_DD_60, CHAIKIN_MF 均属"波动率/风险/资金流"大类，可能存在多重共线性，需 VIF 测试
- **缺失值**: MAX_DD_60 需要 60 天启动数据，新上市/数据短的标的无法计算
- **IC 衰减**: 所有单因子 ICIR 均低于 0.2（28只大盘股上 expected），ML 组合弱信号是核心价值
- **回测过拟合风险**: 已完成全部验证，模型有真实预测能力（见 §7）
- **模型区分度不足**: 模型几乎总是预测正收益（99.6% 的 `pred > 0`），需提升横截面区分度
