## 五、回测验证状态（v0.5.0）

| 验证项 | 结果 | 说明 |
|:---|---:|:---|
| 子区间交叉验证 | ✅ 全部通过 | 3/3 子区间夏普 > 10 |
| 随机标签置换 | ✅ 通过 | p < 0.01，100 次打乱全部超越 |
| 随机策略基线 | ✅ 通过 | p < 0.01，50 次随机全部超越 |
| 费用模型 | ✅ 正确 | vectorbt 费用计算正确，~1.6% 损耗 |
| 安全杠杆 | ✅ 完成 | 保守建议 ≤ 7x |

### ⚠️ 关键发现：收益驱动力

随机基线夏普约 **3.58**，仅略低于模型的 **3.80**。说明：
- **主要驱动力是重新平衡时机**（在牛市中频繁买入），而非个股选择能力
- **模型个股选择增加约 6% 的夏普**（3.58→3.80），虽小但统计显著
- 模型几乎总是预测正收益（99.6% 的 `pred > 0`），区分度不足
- **需要更多横截面区分度**来提高个股选择带来的超额收益
